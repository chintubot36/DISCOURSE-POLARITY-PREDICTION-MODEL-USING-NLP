# Product Review Sentiment Based Ranking System

## 📌 Overview
This project is a Java-based application that analyzes product reviews using sentiment analysis and ranks products based on user feedback.

## 🚀 Features
- Sentiment Analysis using NLP
- Product Ranking based on reviews
- DAO Architecture implementation
- MySQL Database Integration

## 🛠️ Technologies Used
- Java
- Maven
- MySQL
- NLP (Stanford CoreNLP)

## 📂 Project Structure
- DAO Layer
- Service Layer
- Database Layer
- NLP Processing

## ▶️ How to Run
1. Clone the repository
2. Import into IDE (IntelliJ / Eclipse)
3. Setup MySQL database
4. Run MainApp.java

- Add UI (React)
